### CVD 계산 방법 & 코드

#### CVD란?
```
CVD = Cumulative Volume Delta
    = 누적 (매수 체결량 - 매도 체결량)
양수: 매수세 우세 / 음수: 매도세 우세
```

#### Binance 캔들 API에서 CVD 추출
```javascript
// Binance 캔들 데이터 인덱스:
// [0] openTime
// [1] open  [2] high  [3] low  [4] close
// [5] volume (총 거래량)
// [9] takerBuyBaseAssetVolume (시장가 매수 체결량)

async function getCVD(symbol, interval, limit) {
  const url = `https://api.binance.com/api/v3/klines`
            + `?symbol=${symbol}&interval=${interval}&limit=${limit}`
  const klines = await fetch(url).then(r => r.json())

  let cvd = 0
  return klines.map(k => {
    const takerBuy  = parseFloat(k[9])
    const totalVol  = parseFloat(k[5])
    const takerSell = totalVol - takerBuy
    const delta     = takerBuy - takerSell
    cvd += delta
    return {
      time:   k[0],
      open:   parseFloat(k[1]),
      high:   parseFloat(k[2]),
      low:    parseFloat(k[3]),
      close:  parseFloat(k[4]),
      volume: totalVol,
      delta,
      cvd     // 누적값
    }
  })
}
```

#### 선물 CVD (현물 vs 선물 비교)
```javascript
// 선물: fapi.binance.com
const futureCVD = await getCVD(symbol+'USDT', interval, limit)
// 현물: api.binance.com
const spotCVD   = await getCVD(symbol+'USDT_spot', interval, limit)

// 선물 CVD > 현물 CVD → 레버리지 매수세 강함
```

### useApi 훅
```typescript
// hooks/useApi.ts
import { useState, useEffect } from 'react'

export function useApi<T>(url: string) {
  const [data, setData]       = useState<T | null>(null)
  const [loading, setLoading] = useState(true)
  const [error, setError]     = useState<string | null>(null)

  useEffect(() => {
    setLoading(true)
    fetch(url)
      .then(r => r.json())
      .then(d => { setData(d); setLoading(false) })
      .catch(e => { setError(e.message); setLoading(false) })
  }, [url])

  return { data, loading, error }
}
```


```
  Day 3: 탭 네비게이션 + 다크 테마

  Day 1~2: Express 서버 + Railway 배포
  Day 3: CoinGecko + Yahoo Finance 연동
  Day 4: Binance OI/FR/롱숏 연동

  Day 1~2: Binance OI/FR 히스토리 수집 + DB 저장

  Day 5: 예측 모델 (유사 패턴 매칭)

WEEK 4 — 추가 탭
  Day 1~2: 거래량 트래커 (업비트 + KRX + RSI)
  Day 3: FRED 유동성 (TGA/M2)
  Day 4: CVD 스크리너
  Day 5: Hyperliquid 고래

WEEK 5 — 완성
  반응형 모바일
  캐싱 최적화
  리서치 탭 콘텐츠 작성
  Vercel 프론트 배포 + 커스텀 도메인
```

## 더 추가할 수 있는 기능

```
텔레그램 알림 봇
  → OI 급등 / 예측 발표 / 공포탐욕 극단값 시 자동 알림
  → node-telegram-bot-api 사용

BTC ETF 순유입
  → farside.co.uk 크롤링 (현재 사이트에도 미연동 표시)

온체인 강화 (Glassnode 일부 무료)
  → SOPR, NUPL, 거래소 잔고 변화

알트코인 SPF 확장
  → 현재 BTC만 → ETH/SOL/INJ SPF 추가

뉴스 피드
  → CryptoPanic API (무료)
  → https://cryptopanic.com/api/v1/posts/?auth_token={KEY}

사용자 즐겨찾기
  → localStorage 저장 (로그인 없이)
```
