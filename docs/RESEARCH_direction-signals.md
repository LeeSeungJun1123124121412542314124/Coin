작성일: 2026-06-13
상태: 검증 완료 (OOS 통과)
관련: 이 문서는 [SPEC_direction-model.md](SPEC_direction-model.md)의 HA/HMA 방향 모델을 **데이터로 반증·대체**한다. (feature/direction-model 브랜치의 HA 기반 구현은 폐기 대상 — 아래 발견 1 참조)

## 1. 배경 / 원래 문제

변동성 정기 리포트의 "신호·추천 방향이 실제와 반대/무의미"하다는 불만에서 출발. 초기 가설은 "분석기에 이미 있는 방향성 입력(HA·HMA·FR·flow·MVRV)을 종합해 방향 모델을 만들자"였으나(HA 1차 방향 + 보조 신뢰도), 백테스트 적중률이 코인플립 미만이었다. → **"어느 지표가 실제로 방향을 예측하는가"를 데이터로 검증**하는 방향으로 전환.

## 2. 방법론

- **IC (정보계수)** = `Spearman(지표_t, BTC 미래수익률_{t→t+H})`. 부호=방향, |IC|=예측력. scipy 부재로 **순위의 피어슨**으로 계산.
- **호라이즌** H = 30·90일(매크로/방향), 1~24h(변동성).
- **OOS 검증** = TRAIN(2018~2022) / TEST(2023~2026) 분리. 부호 일치 + |IC|≥0.08이면 생존. 구간별(연도) 분해도 병행.
- **복합 합성** = 생존 신호를 bullish=+로 부호정렬 후 **causal(확장창) z-score**(미래정보 배제)로 동등가중 평균.
- **유의성 주의**: 매크로 시계열은 자기상관이 커서 8년에 독립표본 ~30-50개. IC 0.3도 오차범위 존재 → "정밀 타이머 아닌 국면 tilt"로 해석.

## 5. 최종 아키텍처 — 2축 + 레짐 조절기

```
┌─ 방향 축 (30~90일) ──────────────┐   ┌─ 변동성 축 (인트라데이) ─┐
│ 9팩터 복합 (IC 0.35)             │   │ ATR·BB폭·실현변동성       │
│  매크로4 + 온체인2 + 기술3        │   │ (단기 클러스터링 IC 0.3)  │
│  → 강세/약세/중립 tilt           │   │ → "변동성 경보"           │
└──────────────────────────────────┘   └──────────────────────────┘
              │                                      │
              └──────── BB폭 레짐 조절기 ────────────┘
       (확장=방향 신뢰↑ IC 0.335 / 횡보=신뢰↓ IC 0.229)
```

- **방향과 변동성은 별개 축**이다. 방향 복합은 "오를지/내릴지", 변동성 축은 "크게 움직일지"를 예측한다.
- **BB·MACD·MTF의 올바른 자리**: BB→변동성 축 + 레짐 조절기(방향 팩터로는 ✗). MACD→기존 모멘텀과 중복(✗). 주봉·월봉 맥락→**매크로가 이미 담당**(상위 타임프레임 = 월간 유동성·분기 금리). 즉 복합은 이미 멀티타임프레임.

## 6. 9팩터 복합 스펙 (구현용 — 정확한 정의)

각 팩터를 **bullish=+** 로 부호정렬 → causal 확장창 z-score(min_periods=250) → 9개 동등가중 평균 = `composite`.

| # | 카테고리 | 팩터 | 정의 | 부호정렬 | raw IC90 |
|---|---|---|---|---|---|
| 1 | 매크로 | 순유동성 | (WALCL−TGA−RRP×1000) 13주(91일) 변화 | + 그대로 | +0.20 |
| 2 | 매크로 | 달러 | DXY(DTWEXBGS) 13주 변화 | **−** 부호반전 | −0.36 |
| 3 | 매크로 | 금리 | 10년물(DGS10) 13주 변화 | **−** 부호반전 | −0.31 |
| 4 | 매크로 | VIX | VIXCLS 레벨 | + 그대로 | +0.13 |
| 5 | 온체인 | MVRV | CapMVRVCur 레벨 | **−** 부호반전 | −0.14 |
| 6 | 온체인 | 활성주소 | AdrActCnt 13주 변화 | + 그대로 | +0.11 |
| 7 | 기술 | RSI | RSI(14) 일봉 | + 그대로 | +0.15 |
| 8 | 기술 | SMA50이격 | close/SMA50 − 1 | + 그대로 | +0.14 |
| 9 | 기술 | 모멘텀 | close/close.shift(30) − 1 | + 그대로 | +0.13 |

- 단위: WALCL·TGA 백만$, RRP 십억$(×1000으로 맞춤).
- 주간/월간 시리즈는 일봉 인덱스에 ffill.
- `composite > 0` = 강세 tilt, `< 0` = 약세 tilt, 절댓값 = 신뢰도.

## 7. 검증 결과

**복합 IC (vs BTC 미래수익률):**
| 신호 | IC30 | IC90 | TRAIN90 | TEST90 |
|---|---|---|---|---|
| 단일최강 달러(−) | +0.215 | +0.381 | +0.448 | +0.248 |
| 9팩터 복합 | +0.272 | +0.353 | +0.380 | +0.347 |

→ 복합은 피크는 낮지만 **OOS 강건**(TEST에서 달러 0.25 < 복합 0.35).

**복합 tilt 적중률 (90일):**
| 상태 | n | 90d 평균 | 상승률 |
|---|---|---|---|
| 복합 > 0 (강세) | 1,389 | +31.4% | 70.8% |
| 복합 < 0 (약세) | 1,357 | +1.5% | 43.3% |
| (기준) | | | 57.2% |

**레짐별(BB폭):** 확장 IC 0.335(갭 24p) > 횡보 IC 0.229(갭 17p) — 추세장에서 방향 신뢰↑.

## 8. 기각된 후보 (재추가 금지 — 검증 완료)

| 후보 | 기각 사유 |
|---|---|
| HA 캔들·HMA 크로스 | 방향 IC ≈ 0 (8일 상승장 우위는 오버피팅) |
| MACD (시간봉·일봉·주봉) | 방향 IC ~0.05, 기존 모멘텀과 중복 |
| TGA YoY / "BTC 밑" 규칙 | spurious 추세 정렬 / 동행·후행 |
| M2 YoY | IC ≈ 0 |
| 나스닥 모멘텀, 2s10s | OOS 부호 뒤집힘 |
| 신용스프레드(HY) | 데이터 짧음(train 없음) — OOS 불가 |
| 실질금리Δ, 기대인플레 | 금리와 대수적 중복 → 추가 시 과최적화(TEST↓) |
| 거래소 순유출입 | OOS 부호 뒤집힘 |
| 해시레이트Δ | 약함(TEST ≈ 0) |
| 주봉/월봉 RSI·모멘텀 | 일봉·매크로와 중복 → 추가 시 TEST↓ |

> **메타 교훈**: 후보를 추가할 때마다(신규 매크로 / MACD·MTF) in-sample은 좋아 보였으나 **OOS가 일관되게 악화**됐다. "더 많은 지표 ≠ 더 좋음." 9팩터가 검증된 천장.

## 9. 정직한 한계

- 매크로 자기상관 → 독립표본 제한. IC 0.35는 **30~90일 국면 tilt**, 정밀 진입 타이머 아님.
- BTC 단일 자산 중심(기술 스크리닝만 BTC/ETH/SOL 풀링).
- 레짐 의존(추세장 강, 횡보장 약). 비대칭(확장=롱 우위).
- 8.5년(2018~2026) = 약 2.5개 사이클. 더 긴 검증이 이상적.

## 10. 봇 통합 경로

1. **방향**: 실패한 HA/HMA를 9팩터 복합으로 교체. 리포트에 "거시·온체인 국면: 강세/약세/중립 (신뢰도)".
2. **변동성 코어**: 기존 ATR/BB 기반 변동성 경보 유지(봇 원래 강점).
3. **조절기**: BB폭 확장/횡보로 방향 신뢰도 가감.
4. **인프라 필요**: FRED(일/주) + CoinMetrics(일) 데이터 일일 수집 잡 + Avast/TLS 우회(SChannel 또는 CA 번들). 매크로는 일 1회 갱신으로 충분.
